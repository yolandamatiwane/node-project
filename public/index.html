USERS:

HTTP Methods Path Description
GET /users Fetches all users' data
GET /users/:id Fetches single user's data
POST /users/register Adds/registers user to database
POST /users/login Logs in user and returns token
PATCH /users/update/:id Updates user information
DELETE /users/delete/:id Deletes user data

PRODUCTS:

HTTP Methods Path Description
GET /products This returns all products
GET /products/:id Returns one product by specified id
GET /products/recent Returns most recent products
POST /products/addProduct Adds product to database
PATCH /products/update/:id Updates product's information
DELETE /products/delete/:id Deletes product from database
